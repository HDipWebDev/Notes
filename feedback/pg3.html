Image distortion creates a very amateur site. Most people will pick it up immediately. There are a number of ways to ensure that images maintain their aspect ratio

  • Only specify one dimension, not both

Sprites

Sprites - Additional Information

Sprites are a simple way to cut down traffic on your server. If your site uses icon or other GUI components, you may find it easier to implement sprites

CSS sprites typically comprise a single image file for multiple GUI components. When used, the developer only shows a small portion of the image in question

The use of sprites significantly cuts down traffic on the server. Normally each component request, generates a request on the server. With sprites, only one request is needed; CSS takes over from there.
